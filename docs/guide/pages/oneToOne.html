3.1 One-To-One Relations

Given the following the following one-to-one relationship: 
class Bar {
   String bar
   static mapping = {
      datasource = 'lookup'
   }
}
class Foo {
   String foo
   @Relation
   Bar bar
   static mapping = {
      datasource = 'audit'
   }
}

The transformation will modify Foo.class to the following: 

class Foo {
   String foo
   @Relation
   Bar bar
   Long barId
   static transients = [ 'bar' ]
   static mapping = {
      datasource = 'audit'
   }
   public Bar getBar() {
      if(barId) {
         return Bar.get(barId)
      } else {
         return null
      }
   }
   public void setBar(Bar bar) {
      if(bar) {
         barId = bar.id
      }
   }
}
